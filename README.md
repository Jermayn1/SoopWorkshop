# SoopWorkshop
Automatisches Auswertungstool für Java-Aufgaben im SOOP-Workshop. Entwickelt als Projekt neben der Ausbildung.

---

Alles untenstehende sind meine Pläne für das Projekt. In laufe der Umsetzung des Projekts könnte sich die Features und die Struktur anpassen.

---

## Über dieses Projekt

Dieses Projekt entsteht neben meiner Ausbildung zum Fachinformatiker für Anwendungsentwicklung als eigenständiges Lernprojekt. Ich nutze es, um meine Kenntnisse in C#, Blazor und Clean Architecture praktisch zu vertiefen und gleichzeitig ein Tool zu bauen, das ich wirklich einsetze.

---

## Geplante Features


- 📋 Aufgabenliste — Sidebar mit allen Workshop-Aufgaben, Klick öffnet die Aufgabenstellung
- 📤 JAR-Upload — Teilnehmer laden ihre kompilierte Java-Lösung direkt hoch
- ⚙️ Automatische Auswertung — Das Tool führt das JAR aus und bewertet nach Kategorien
- 📊 Detailliertes Feedback — Ergebnis pro Kategorie (Clean Code, Testfälle, etc.)
- 🔒 Sandbox-Ausführung — Jedes JAR läuft isoliert in einem Docker-Container
- ⚡ Asynchrone Verarbeitung — Upload gibt sofort eine ID zurück, Ergebnis wird gepollt
